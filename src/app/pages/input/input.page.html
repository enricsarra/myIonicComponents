Component ion-input ion-input The input component is a wrapper to the HTML input element with custom styling and additional functionality. It accepts most of the same properties as the HTML input, but works great on desktop devices and integrates with the keyboard on mobile devices. It is meant for text type inputs only, such as "text", "password", "email", "number", "search", "tel", and "url".
