Component ion-refresher ion-refresher The refresher provides pull-to-refresh functionality on a content component. The pull-to-refresh pattern lets a user pull down on a list of data using touch in order to retrieve more data. Data should be modified during the refresher's output events. Once the async operation has completed and the refreshing should end, call complete() on the refresher.
